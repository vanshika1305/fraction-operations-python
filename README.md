# Fraction Operations using Python (OOP)

This project implements a Fraction class in Python using Object-Oriented Programming concepts and magic methods.

## Features

* Addition of fractions
* Subtraction of fractions
* Multiplication of fractions
* Division of fractions
* Automatic simplification using GCD
* Operator overloading using magic methods

## Concepts Used

* OOP (Classes & Objects)
* Magic Methods (**add**, **sub**, etc.)
* Encapsulation
* Python math module

## Example

```python
f1 = Fraction(1, 2)
f2 = Fraction(3, 4)

print(f1 + f2)  # Output: 5/4
```
